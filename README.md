# Vatsim-airport-traffic

TypeScript CLI for displaying live VATSIM airport traffic.

## Installation

Install the project dependencies:

```text
npm install
```

## Usage

Run the CLI with an airport code:

```text
npm start -- EIDW
```

Replace `EIDW` with the ICAO code of the airport you want to check.

Example output:

```text
Departures:
VJT565 EIDW -> HESH

Arrivals:
UPS207 KSDF -> EIDW
```
